# v2.0.0 Release Notes

## Summary

`v2.0.0` is the first cleanup release after the sync/backfill work. The main goal is to make the CLI surface match the actual product, simplify sync behavior, and remove interfaces that were drifting or no longer justified.

## Breaking Changes

- Removed the top-level `watch` command.
- Removed the sync TUI and switched sync to plain terminal progress output.
- Removed dead CLI surface that was still appearing in help/docs:
  - global `--format`
  - `sync run --simple`
- Sync/storage wording was tightened:
  - `--storage` is now the primary flag
  - hidden `--db` compatibility remains only as an alias

## User-Facing Changes

- Sync now ends with a visible plain-text summary instead of auto-closing a TUI.
- `sync status` reports `performance_metrics` and points users to the correct `garmin sync run` command.
- Homebrew/release publishing is more robust:
  - release artifacts produce `checksums.txt`
  - the Homebrew formula is generated from release checksums
  - manual release reruns target the requested tag instead of `main`

## Data and Sync Improvements

- Added pipeline-aware backfill state and improved sync scheduling.
- Fixed backfill for daily health and performance metrics.
- Skips duplicate sync tasks.
- Fixed sync status profile resolution to use the latest synced profile.
- Fixed parsing of `HRV lastNightAvg` for daily health.

## Upgrade Notes

- If you were using `garmin watch`, switch to `garmin sync run`.
- If you relied on the previous TUI behavior, expect plain-text sync progress instead.
- If you scripted removed flags, update those scripts before adopting `v2.0.0`.

## Suggested Release Checklist

1. Wait for the manual `v1.0.6` release rerun to finish and verify Homebrew is repaired.
2. Review this release note and edit the wording you want on the GitHub release page.
3. Tag the release:

```bash
git tag -a v2.0.0 -m "v2.0.0"
git push origin v2.0.0
```

4. Confirm the `release` workflow publishes artifacts and updates `vicentereig/homebrew-tap`.
5. Verify install paths after the workflow finishes:

```bash
brew install vicentereig/tap/garmin
garmin --help
```
