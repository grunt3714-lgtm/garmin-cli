# v2.0.1 Release Notes

## Summary

`v2.0.1` is a patch release to restore Garmin authentication after Garmin changed the SSO page structure and broke the old HTML-scraping flow.

## User-Facing Changes

- `garmin auth login` now uses Garmin's mobile JSON auth endpoints instead of scraping HTML response pages.
- MFA login continues to work through the same CLI flow.
- CI formatting issues from the auth fix were cleaned up in a follow-up commit before release.

## Known Follow-Up

- Issue #6 tracks the remaining work to make the OAuth handoff fully match `garth`'s current session behavior:
  https://github.com/vicentereig/garmin-cli/issues/6

## Suggested Release Checklist

1. Review this release note and edit the wording you want on the GitHub release page.
2. Tag the release:

```bash
git tag -a v2.0.1 -m "v2.0.1"
git push origin v2.0.1
```

3. Confirm the `release` workflow publishes artifacts, pushes to crates.io, and updates `vicentereig/homebrew-tap`.
4. Verify install paths after the workflow finishes:

```bash
brew install vicentereig/tap/garmin
garmin auth login
garmin --help
```
